# Доработки 

- На странице списка проектов на Windows может появляться некрасивый скроллинг на всю страницу. 
- После добавления урока на сервер не обновляются статусы обработки уроков в шаблоне. 
- Если добавить урок в программе, а потом удалить его на сервере, то в программе он остается в списке уроков даже после перезапуска программы. 

## Что можно добавить

- На странице списка проектов справа добавить отображение локальной очереди обработки.
