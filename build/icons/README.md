# Иконки Video2Book

## Исходный файл

Поместите ваш PNG файл размером 1024x1024 пикселей в эту папку с именем `icon.png`.

## Автоматическая генерация

После размещения `icon.png` выполните команду:

```bash
npm run build:icons
```

Это создаст следующую структуру:

```
build/icons/
├── icon.png (исходный файл 1024x1024)
├── mac/
│   └── icon.icns (для macOS)
├── win/
│   └── icon.ico (для Windows)
└── png/
    ├── 16x16.png
    ├── 24x24.png
    ├── 32x32.png
    ├── 48x48.png
    ├── 64x64.png
    ├── 128x128.png
    ├── 256x256.png
    └── 512x512.png
```

## Использование

Иконки автоматически используются при сборке приложения:
- macOS: `build/icons/mac/icon.icns`
- Windows: `build/icons/win/icon.ico`
- Linux: PNG файлы из `build/icons/png/`

## Требования к исходной иконке

- Формат: PNG
- Размер: 1024x1024 пикселей
- Фон: прозрачный
- Стиль: простой, хорошо читаемый на малых размерах
